# Écriture Réseaux Sociaux

> Écriture pour YouTube, Instagram et TikTok : hooks, rétention, scripts visuels, formats algorithmiques.

## When to Use

- Writing video scripts for YouTube, Instagram Reels, or TikTok
- Crafting hooks that stop the scroll
- Structuring retention curves for different platforms
- Converting a brand or creative concept into platform-native content
- Planning visual-first content where script is secondary to image
- Extends `ecriture-script` (tronc commun) with social media-specific craft

## How It Works

### 1. The Hook — Première Seconde

The hook is the single most important element on social platforms. The audience decides in 1-3 seconds.

**Hook types:**

| Type | Example | Best for |
|------|---------|----------|
| Question | "Tu savais que 80% des vidéos corporate sont ignorées?" | Educational, B2B |
| Statement choc | "J'ai filmé 200 mariages. Celui-ci m'a fait pleurer." | Story, emotional |
| Visual grab | [Close-up of unexpected action — no words needed] | Reels, TikTok |
| Pattern interrupt | Familiar setup → unexpected twist in first 2 seconds | TikTok, viral |
| Promise | "En 60 secondes, je te montre comment filmer comme un pro avec ton iPhone" | Tutorial, value |
| Mid-action | Start in the middle of the most intense moment | Vlog, behind-scenes |

**Lavandier applied to hooks:** The hook IS dramatic irony compressed to 1 second — the audience glimpses a tension they need to see resolved.

### 2. Retention Structure

#### YouTube (2-15 min)

```
0:00-0:03  HOOK — Promise or question
0:03-0:30  CONTEXT — Why this matters, establish credibility
0:30-2:00  CHAPTER 1 — First value delivery (re-hook at each chapter)
2:00-5:00  CHAPTER 2 — Deepest value (the reason they stay)
5:00-7:00  CHAPTER 3 — Application or story payoff
7:00-end   CTA — Subscribe, comment, next video tease

RE-HOOKS every 30-60s: mini-cliffhangers, "but here's the thing...", visual changes
```

**Truby applied to YouTube:** The video has a moral argument. The viewer starts with a belief (weakness), the content challenges it, and by the end they see differently (self-revelation).

#### Instagram Reels (15-60s)

```
0:00-0:02  HOOK — Visual or text overlay, must work on mute
0:02-0:05  SETUP — Context in 3 seconds
0:05-0:20  CONTENT — One idea, one technique, one story beat
0:20-0:28  PAYOFF — The result, the punchline, the transformation
0:28-0:30  CTA or LOOP — End feeds back to start for replay

KEY RULE: Must work WITHOUT sound. Text overlays, visual storytelling.
```

#### TikTok (15-60s)

```
0:00-0:01  HOOK — Pattern interrupt or trending sound cue
0:01-0:05  SETUP — Relatable situation or premise
0:05-0:20  BUILD — Escalation, tension, anticipation
0:20-0:28  PAYOFF — Satisfying resolution or surprise
0:28-0:30  OPEN LOOP — Leave something unresolved for comments/follows

KEY RULE: Authenticity > production value. Overly polished = scroll past.
```

### 3. Script vs Visual Structure

Not all social content needs a written script. Choose the right approach:

#### Full Script

When to use: tutorials, educational content, client brand content
```
[TEXTE À L'ÉCRAN]: "3 erreurs de cadrage"
[VOIX OFF]: "La première erreur que je vois partout..."
[ACTION]: Montre l'erreur sur un exemple concret
[TEXTE]: "ERREUR #1: Trop d'air au-dessus de la tête"
```

#### Visual Storyboard (No Script)

When to use: aesthetic content, behind-scenes, process videos
```
PLAN 1 (0:00-0:03) — Mains sur la caméra, gros plan, lumière naturelle
PLAN 2 (0:03-0:06) — Le sujet entre dans le cadre, rack focus
PLAN 3 (0:06-0:10) — Détail du travail en cours, mouvement lent
PLAN 4 (0:10-0:15) — Reveal du résultat final, plan large
MUSIQUE: [Trending sound or ambient]
TEXTE: Minimal — just the CTA at the end
```

#### Hybrid (Script + Improv Zones)

When to use: personality-driven content, vlogs, talking-head with B-roll
```
[SCRIPT]: "Aujourd'hui je vous emmène sur un tournage brand content à 750€"
[IMPROV ZONE]: Réaction naturelle en arrivant sur le lieu
[SCRIPT]: "Voici les 3 choses que je prépare toujours avant d'appuyer sur REC"
[IMPROV ZONE]: Montrer le setup réel, commenter en live
[SCRIPT]: CTA final
```

### 4. Platform-Specific Rules

| Rule | YouTube | Instagram | TikTok |
|------|---------|-----------|--------|
| Optimal duration | 8-12 min | 30-45s | 15-30s |
| Sound importance | Critical | Optional (captions) | Critical (trending) |
| Production value | High expected | Medium-high | Low preferred |
| CTA placement | End + pinned comment | Bio link + caption | Comment + profile |
| Algorithm signal | Watch time % | Saves + shares | Completions + replays |
| Posting frequency | 1-2/week | 4-7/week | 1-3/day |
| Text overlays | Thumbnails matter most | Essential (mute viewing) | Common (trending formats) |

### 5. CTA Placement

**YouTube:** "Si cette vidéo t'a aidé, abonne-toi" — after value delivery, not before
**Instagram:** In caption + bio link — keep the Reel itself clean
**TikTok:** In comments (pinned) or as a follow-up video — never interrupt the flow

## Examples

### YouTube Script — Behind the Scenes Brand Content

```
HOOK (0:00-0:05):
  [Plan: Richard arrive devant l'atelier, caméra à la main]
  VOIX: "750 euros, 5 livrables, une journée. Voici comment je fais."

CHAPITRE 1 — LA PRÉPA (0:05-2:00):
  [Plans: ordinateur avec le séquencier, moodboard, checklist matériel]
  VOIX: "Avant d'appuyer sur REC, j'ai déjà tout écrit..."
  RE-HOOK (1:30): "Mais le truc qui change tout, c'est ça..."

CHAPITRE 2 — LE TOURNAGE (2:00-6:00):
  [Plans: setup lumière, interview, plans détail, problème imprévu]
  VOIX: Narration du processus + moments authentiques
  RE-HOOK (4:00): "Et là, le client me dit un truc que j'avais pas prévu..."

CHAPITRE 3 — LE RÉSULTAT (6:00-8:00):
  [Plans: montage accéléré, before/after, réaction client]
  VOIX: "Voilà ce que ça donne..."

CTA (8:00-8:30):
  VOIX: "Si tu veux voir le séquencier que j'utilise, lien en description."
```

### Instagram Reel — Quick Tip

```
[0:00] TEXTE: "L'erreur #1 en vidéo corporate"
       VISUEL: Plan raté (trop sombre, mal cadré)
[0:03] TEXTE: "Tu filmes le produit"
       VISUEL: Zoom produit seul, plat, ennuyeux
[0:06] TEXTE: "Filme la personne qui l'utilise"
       VISUEL: Même produit, mais avec une main, un sourire, une émotion
[0:12] TEXTE: "L'émotion > la technique"
       VISUEL: Side-by-side comparison
[0:15] LOOP: Cut back to the opening shot
```
